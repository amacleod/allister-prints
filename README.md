Allister's Downloaded Prints
============================

Originally I just started naming folders to include the author's name,
like `Gridfinity_by_Zack_Freedman`, but that might not be enough
information to reliably identify a print in the future, when squishy
human memory has slumped. To address that, I want to start including
hyperlinks to the entries on Printables, MakerWorld, Thangs,
Thingiverse, and so on.
